# f1-race-strategy-analysis

This project analyzes multi-season Formula 1 race data to explore how race strategy,
pit stop timing, and track characteristics influence performance outcomes.

## Status
This project is currently under active development.

## Tools
- Python (Pandas, NumPy)
- SQL (SQLite)
- Plotly
- Streamlit
- REST APIs (Ergast API)

## Goals
- Ingest and clean multi-season Formula 1 race data
- Perform exploratory data analysis and feature engineering
- Analyze relationships between strategy decisions and race outcomes
- Build interactive dashboards to visualize performance trends
